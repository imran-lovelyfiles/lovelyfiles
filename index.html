Why is this free?

Because it costs nothing to run. LovelyFiles has no servers — every tool works inside your own browser using open web technology. Your files are processed on your device and never travel anywhere. No accounts, no limits, no ads. The code is open source, so anyone can read it, improve it, or host their own copy.
